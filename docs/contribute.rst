============
Contribute
============
DHL-LinAlg can be installed directly via pip from PyPi:

..  code-block:: bash

   pip install dhllinalg 

or you can build from source

..  code-block::
    
    git clone --recurse-submodules https://github.com/DHL-ASC/DHL-LinAlg.git
    cd DHL-LinAlg
    pip install .
